# Шорткаты плеера

Горячие клавиши для страницы просмотра YouTube. Все работают в полноэкранном и обычном режимах. Независимы от раскладки — используется `KeyboardEvent.code`, а не `key`.

## Шорткаты

| Клавиши | Действие | Fullscreen | Обычный |
|---------|----------|:----------:|:-------:|
| `Shift+C` | Открыть/скрыть комментарии / прокрутить к ним | ✓ | ✓ |
| `Shift+L` | Лайк / убрать лайк | ✓ | ✓ |
| `Shift+D` | Дизлайк / убрать дизлайк | ✓ | ✓ |
| `Shift+I` | Открыть/скрыть описание / прокрутить к нему | ✓ | ✓ |
| `Shift+S` | Открыть/скрыть меню настроек, фокус на поле поиска | ✓ | ✓ |

## Поведение

- **Определение fullscreen.** Читается атрибут `is-fullscreen` на панели комментариев. Надёжнее, чем `document.fullscreenElement`, который равен `null`, когда YouTube использует собственную реализацию полноэкранного режима.
- **Поиск кнопок.** Каждый шорткат сначала ищет в `.ytp-fullscreen-quick-actions`, затем в `#actions`. Лайк/Дизлайк ищутся по кастомным элементам-обёрткам (языконезависимо). Кнопка комментариев в fullscreen ищется по `aria-label` с fallback на уникальный SVG path.
- **Действие.** Шорткат кликает нативную кнопку YouTube. Событие подавляется только если цель найдена.
- **Скролл к комментариям (`Shift+C`, обычный режим).** Если `#comments` в верхней половине вьюпорта — прокрутка к началу страницы; иначе — к `#comments`.
- **Описание (`Shift+I`).** Fullscreen: открывает панель структурированного описания, или закрывает, если она уже раскрыта. Обычный режим: раскрывает встроенное описание и прокручивает к нему; если уже видно — к началу страницы.
- **Отступ при прокрутке.** `window.scrollTo` со смещением = актуальная высота `ytd-masthead` + 5px. Считывается на каждом вызове.
- **Настройки (`Shift+S`).** Кликает `.ytp-settings-button` для переключения меню. При открытии запрашивает фокус на поле поиска через `focusPlayerMenuSearch()`.
- **Бейджи в тултипах.** `MutationObserver` патчит общий `yt-popover`. Отслеживание `mouseover` определяет активную кнопку — языконезависимо, работает в любом состоянии кнопки.

## Файлы

- `modules/playerShortcuts.js` — регистрация и поиск кнопок
- `modules/shortcutCore.js` — `registerShortcut`, `initTooltipWatcher`, скролл-хелперы
- `modules/shortcuts.js` — коды клавиш и бейджи
