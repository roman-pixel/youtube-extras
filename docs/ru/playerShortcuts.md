# Шорткаты плеера

## Что делает

Добавляет горячие клавиши для действий с плеером YouTube. Работает в полноэкранном и обычном режиме. Каждый шорткат отображается как бейдж в нативном тултипе соответствующей кнопки.

## Шорткаты

| Клавиши | Действие | Fullscreen | Обычный |
|---------|----------|:----------:|:-------:|
| `Shift+C` | Открыть / скрыть комментарии / прокрутить к ним | ✓ | ✓ |
| `Shift+L` | Лайк / убрать лайк | ✓ | ✓ |
| `Shift+D` | Дизлайк / убрать дизлайк | ✓ | ✓ |
| `Shift+I` | Открыть / скрыть описание / прокрутить к нему | ✓ | ✓ |

Все шорткаты работают на любой раскладке клавиатуры — используется `e.code` (физическая позиция клавиши), а не `e.key` (символ).

## Как работает

**Определение полноэкранного режима**
YouTube добавляет атрибут `is-fullscreen` на элемент панели комментариев при входе в fullscreen. Это надёжнее, чем `document.fullscreenElement`, который возвращает `null` когда YouTube использует собственную реализацию полноэкранного режима.

**Поиск кнопок**
Каждый шорткат ищет кнопку последовательно в двух контекстах: панель быстрых действий fullscreen (`.ytp-fullscreen-quick-actions`) и панель действий обычной страницы просмотра (`#actions`). Лайк и дизлайк ищутся по кастомным элементам-обёрткам (`like-button-view-model`, `dislike-button-view-model`) — они языконезависимы. Кнопка комментариев в fullscreen ищется по `aria-label` с fallback на уникальный SVG path иконки.

**Триггер действий**
Каждый шорткат кликает нативную кнопку YouTube, не манипулируя DOM-состоянием напрямую. Это сохраняет консистентность внутреннего состояния YouTube (загрузка контента, анимации, счётчики). Событие клавиши подавляется только если цель найдена — иначе оно передаётся YouTube как обычно.

**Комментарии в обычном режиме**
Вне fullscreen `Shift+C` переключает между секцией комментариев и началом страницы. Если `#comments` уже находится в верхней половине вьюпорта (пользователь долистал до комментариев), шорткат прокручивает назад к `#primary`. Иначе — прокручивает вниз к `#comments`.

**Описание (`Shift+I`)**
В fullscreen открывает панель структурированного описания через `yt-player-overlay-video-details-renderer`. Если панель уже раскрыта (`visibility="ENGAGEMENT_PANEL_VISIBILITY_EXPANDED"`), она закрывается через её кнопку видимости. В обычном режиме шорткат раскрывает встроенное описание (`#description-inline-expander #expand`) и прокручивает к нему; если описание уже в верхней половине вьюпорта — прокручивает обратно к `#primary`.

**Бейджи в тултипах**
YouTube использует один общий элемент `yt-popover` для тултипов всех кнопок. `MutationObserver` следит за ним. Отслеживание hover через `mouseover` определяет, над какой кнопкой находится курсор — поэтому бейджи вставляются независимо от языка и состояния кнопки (например, «Нравится» vs «Больше не нравится»). Бейджи работают в обоих режимах.
