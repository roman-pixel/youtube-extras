# Шорткат комментариев

## Что делает

Добавляет шорткат `Shift+C` для открытия и закрытия панели комментариев в полноэкранном режиме YouTube. Также патчит тултип кнопки, добавляя подсказку о шорткате.

## Шорткат

| Клавиши | Действие |
|---------|----------|
| `Shift+C` | Открыть / скрыть комментарии (только в fullscreen) |

Работает на любой раскладке клавиатуры — используется `e.code` (физическая позиция клавиши), а не `e.key` (символ).

## Как работает

**Определение полноэкранного режима**
YouTube добавляет атрибут `is-fullscreen` на элемент панели комментариев при входе в fullscreen. Это надёжнее, чем `document.fullscreenElement`, который возвращает `null` когда YouTube использует собственную реализацию полноэкранного режима.

**Поиск кнопки**
Кнопка комментариев находится внутри `.ytp-fullscreen-quick-actions`. Сначала ищется по `aria-label` (RU и EN локали), затем по уникальному SVG path иконки как языконезависимый fallback.

**Триггер действия**
Вместо прямой манипуляции атрибутами DOM модуль кликает нативную кнопку YouTube. Это гарантирует загрузку контента комментариев и сохраняет консистентность внутреннего состояния YouTube.

**Патч тултипа**
YouTube использует один общий элемент `yt-popover` для тултипов всех кнопок. `MutationObserver` следит за ним и добавляет стилизованный бейдж `Shift+C` когда активен тултип кнопки комментариев. Существующий текстовый узел сохраняется, чтобы YouTube мог продолжать обновлять его для других кнопок.
