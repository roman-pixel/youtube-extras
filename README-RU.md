# YouTube Extras

Chrome-расширение. Добавляет горячие клавиши и поиск в меню настроек плеера.

## Шорткаты

Все шорткаты работают в полноэкранном и обычном режимах. Каждая комбинация отображается как бейдж в нативном тултипе кнопки.

| Шорткат | Действие |
|---------|----------|
| `Shift+C` | Открыть/скрыть комментарии / прокрутить к ним |
| `Shift+L` | Лайк / убрать лайк |
| `Shift+D` | Дизлайк / убрать дизлайк |
| `Shift+I` | Открыть/скрыть описание / прокрутить к нему |
| `Shift+S` | Открыть/скрыть меню настроек (автофокус в поле поиска) |
| `Shift+F` | Фокус в поле поиска внутри открытого меню настроек |
| `Enter` / `Shift+Enter` | Следующее / предыдущее совпадение в поиске |
| `Escape` | Снять фокус с поля поиска |

## Поиск в меню

В меню настроек плеера (`.ytp-settings-menu`) появляется строка поиска, если в панели больше 5 пунктов: субтитры, авто-перевод и т. п. Подстроковый фильтр, счётчик совпадений, навигация вперёд/назад, прокрутка к текущему совпадению. Тексты UI локализованы — русский для браузерных локалей `ru-*`, английский для остальных.

## Установка

1. Склонировать репозиторий
2. Открыть `chrome://extensions/`
3. Включить режим разработчика
4. Нажать «Загрузить распакованное» → выбрать папку проекта

## Структура проекта

```
youtube-extras/
├── manifest.json
├── content.js                  # Точка входа
├── styles/
│   ├── badge.css               # Бейдж шортката
│   ├── tooltip.css             # Кастомный тултип
│   └── search.css              # Строка поиска в меню
└── modules/
    ├── youtubeSelectors.js     # DOM-селекторы YouTube
    ├── shortcuts.js            # Коды клавиш и бейджи
    ├── i18n.js                 # Строки для пользователя (ru/en)
    ├── shortcutCore.js         # registerShortcut, tooltip watcher, скролл-хелперы
    ├── search.js               # Универсальный инжектор поиска
    ├── playerShortcuts.js      # Шорткаты плеера
    ├── playerMenuSearch.js     # Подключение поиска к меню настроек плеера
    └── debugAutoHide.js        # Отключает автоскрытие плеера (debug, по флагу)
```
