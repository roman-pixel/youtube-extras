# YouTube Extras

Chrome-расширение, которое добавляет недостающие горячие клавиши и улучшения интерфейса YouTube.

## Возможности

Все шорткаты работают только в полноэкранном режиме и отображаются как бейджи в нативных тултипах кнопок.

| Шорткат | Действие |
|---------|----------|
| `Shift+C` | Открыть / скрыть комментарии |
| `Shift+L` | Лайк / убрать лайк |
| `Shift+D` | Дизлайк / убрать дизлайк |

## Установка

1. Скачать или склонировать репозиторий
2. Открыть Chrome и перейти на `chrome://extensions/`
3. Включить **Режим разработчика** (переключатель справа вверху)
4. Нажать **Загрузить распакованное расширение** и выбрать папку проекта
5. Открыть YouTube, запустить видео, войти в полноэкранный режим и попробовать шорткаты

## Структура проекта

```
youtube-extras/
├── manifest.json
├── content.js                  # Точка входа — инициализирует все модули
├── styles/
│   └── badge.css               # Стили бейджа шортката, инжектируемые на страницы YouTube
└── modules/
    ├── shortcutCore.js         # Общие утилиты (определение fullscreen, тултипы, регистрация шорткатов)
    ├── playerShortcuts.js      # Все шорткаты плеера
    └── debugAutoHide.js        # Отключает автоскрытие контролов (активируется через localStorage)
```

## Участие в разработке

Pull request'ы приветствуются. При добавлении новой функции:
1. Создать новый файл в `modules/`
2. Использовать `registerFullscreenShortcut()` и `initTooltipWatcher()` из `shortcutCore.js`
3. Вызвать функцию инициализации из `content.js`
4. Добавить документацию в `docs/en/` и `docs/ru/`
