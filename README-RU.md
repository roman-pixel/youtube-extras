# YouTube Extras

Chrome-расширение, которое добавляет недостающие горячие клавиши и улучшения интерфейса YouTube.

## Возможности

| Шорткат | Действие | Контекст |
|---------|----------|----------|
| `Shift+C` | Открыть / скрыть комментарии | Полноэкранный режим |

Больше функций в разработке.

## Установка

1. Скачай или склонируй репозиторий
2. Открой Chrome и перейди на `chrome://extensions/`
3. Включи **Режим разработчика** (переключатель справа вверху)
4. Нажми **Загрузить распакованное расширение** и выбери папку проекта
5. Открой YouTube, запусти видео, войди в полноэкранный режим и нажми `Shift+C`

## Структура проекта

```
YT_extension/
├── manifest.json
├── content.js              # Точка входа — инициализирует все модули
└── modules/
    └── commentsShortcut.js # Шорткат Shift+C + патч тултипа
```

## Участие в разработке

Pull request'ы приветствуются. При добавлении новой функции:
1. Создай новый файл в `modules/`
2. Объяви функцию `initYourFeature()` на верхнем уровне
3. Вызови её из `content.js`
4. Добавь документацию в `docs/en/` и `docs/ru/`
